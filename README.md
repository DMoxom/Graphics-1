# Graphics-1
Graphics 1 University assignment. Overall grade 75%.

### Feature Checklist
- Ability to display a 3D model loaded from .MD2 file
- Model can be rotated around X, Y and Z
- Model can be scaled and translated in world space
- Back-face culling
- Polygon sorting
- Flat shading using Windows GDI calls
- Ambient lighting
- Directional lighting source showing diffuse lighting
- Point lighting source showing diffuse lighting
- Flat shading using own rasterization code
- Gouraud shading
